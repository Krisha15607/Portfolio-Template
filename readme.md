# 🌟 Krisha's Portfolio

A modern, animated, and fully responsive portfolio website built with HTML, CSS, and JavaScript. Features beautiful light and dark themes with smooth transitions and glassmorphism effects.

## ✨ Features

- 🎨 **Modern Design** - Beautiful gradient colors and glassmorphism effects
- 🌓 **Light & Dark Theme** - Toggle between themes with smooth transitions
- 📱 **Fully Responsive** - Works perfectly on all devices
- ✨ **Smooth Animations** - Engaging animations and transitions
- 🎯 **Interactive Elements** - Hover effects and dynamic interactions
- 📧 **Contact Form** - Functional contact form with validation
- 🎭 **Skill Showcase** - Animated skill progress bars
- 💼 **Project Gallery** - Beautiful project cards with hover effects

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Variables, Flexbox, Grid
- **JavaScript** - Interactive functionality
- **Font Awesome** - Icons
- **Google Fonts** - Poppins & Playfair Display

## 📋 Sections

1. **Home** - Hero section with animated background
2. **About** - Information about Krisha's education and interests
3. **Skills** - HTML, CSS, JavaScript, and Designing skills
4. **Projects** - Portfolio of recent work
5. **Contact** - Get in touch form

- Transparent backgrounds with blur effects
- Modern card designs
- Elegant navigation bar

## 📝 Customization

To customize the portfolio:

1. **Colors**: Edit CSS variables in `style.css`
2. **Content**: Update text in `index.html`
3. **Images**: Replace placeholder images with actual photos
4. **Social Links**: Update href attributes in social link sections
